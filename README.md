# MovieTV
Esta aplicación utiliza la API de [TheMovieDB](https://www.themoviedb.org/) para obtener información sobre películas y series de televisión.

# Como usar
```
git clone https://github.com/Ulian/movietv.git
cd movietv
npm install
cd client
npm install
npm run-script build
cd ..
npm start
```
